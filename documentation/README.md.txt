Swag Labs Cypress Portfolio

Project Overview
This portfolio demonstrates automated end-to-end tests for Swag Labs using Cypress, complementing manual QA workflows.

Features Tested
- Login functionality
- Product sorting (e.g., price, name)
- Cart operations (add, update, remove items)
- Checkout workflow

Tools & Technologies
- Cypress
- Node.js
- GitHub
